  Bitcoin Wallet API Library (Airbitz Core)

The Airbitz Bitcoin wallet and directory is available as a powerful, cross platform API library which includes all the functionality and ease-of-use of the Airbitz mobile application. The Airbitz Core (ABC) gives developers the ability to build Bitcoin products such as currency exchanges, ATMs, POS solutions, games, and payroll services that are fully integrated with accounts and wallets from the Airbitz mobile app. The Airbitz mobile app is the first product utilizing this open platform, and the potential for future integrations with other financial and alternative services is vast.

The Airbitz Bitcoin Wallet is built on the cross platform, C/C++ Airbitz Core library. This library implements the Airbitz wallet functionality in its entirety, minus the graphical interface. The Airbitz Core library makes it possible to quickly build applications where users control their private keys safely and securely without compromising privacy, features, or usability.

The Airbitz Core enables an ecosystem of applications built on the Airbitz platform with both the compatibility and capability to securely access a usersâ€™ funds from within various external applications. A wallet created on the Airbitz mobile application can be quickly shared with a multiplayer game to earn Bitcoin from within the app, a mobile shopping app to pay for goods, or financial planning software to help with budgeting and tracking, all from one account.

Decentralized and Private by Design

The Airbitz Core library can be utilized by other Bitcoin companies and projects. Utilization of the Airbitz Core (ABC) can significantly reduce implementation time for a Bitcoin wallet backend system. ABC implements the following functionality of the Airbitz wallet:

  • User authentication
  • Account/Wallet Creation
  • Client-side Encryption
  • Multi-device synchronization & Backup
  • Transaction Meta-data management
  • Bitcoin Address generation / key management
  • Transaction signing / broadcasting
  • Decentralized access to bitcoin network
  • Shared wallets w/multisig (Q1 2015)
