Airbitz Core Platform

  Airbitz Core Platform

The entire functionality and ease of use provided by the Airbitz Bitcoin wallet and directory is available as a powerful, cross platform library/API. The Airbitz Core (ABC) allows developers to quickly build products such as currency exchanges, ATMs, POS solutions, games, and payroll services and have them fully integrated with accounts & wallets from the Airbitz mobile app. The Airbitz mobile app is the first product utilizing this powerful platform, and the possible integration with other financial services is tremendous.

The Airbitz wallet is built on the cross platform, C/C++ Airbitz Core library. This library alone implements the full functionality of the Airbitz wallet minus the graphical interface. With the Airbitz Core library it is possible to quickly build applications where users to control their private keys safely and securely without compromising privacy, features, or ease of use.

Airbitz Core Bitcoin Wallet Library

Decentralized and Private by Design

The Airbitz Core library can be utilized by other bitcoin companies and projects and significantly reduce their time to implement a bitcoin wallet backend system complete with user authentication, encryption, backup, and meta-data management. This library could be tremendously useful to bitcoin companies such as currency exchanges, ATMs, and payment processors.

  • User authentication
  • Encryption
  • Backup
  • Meta-data management

Use Cases

The Airbitz Core library is designed for Bitcoin projects and any financial service company with the need for Bitcoin wallet functionality while ensuring that users remains in control of their funds.
