Hiddenbits is a citywide scavenger hunt that allows you to collect real money by following the steps below.

After following each clue if you are the first to the location you will find a physical card with bitcoin virtual currency on. Each gift card will have between 0.1 BTC ($20) and 2 BTC ($460) loaded on it.

Using Airbitz bitcoin wallet you will be not only be able to redeem this virtual currency but you will be able to send and receive bitcoin anywhere in the world instantly via 100% securely via email or SMS.

Bitcoin has a variable value but below is a live price with the current value of the virtual currency.
