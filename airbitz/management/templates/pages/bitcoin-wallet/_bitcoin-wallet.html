  Airbitz Bitcoin Wallet

  Airbitz Bitcoin Wallet

No setup or configuration necessary to send or receive bitcoin, anywhere in the world, in just minutes. The Airbitz bitcoin wallet is one of the easiest wallets to start receiving or sending bitcoin and also includes a curated map and directory of both local and web based bitcoin business.

Airbitz wallet has enhanced privacy, security, and features a user-first interface which solves many of the usability issues plaguing current bitcoin wallet solutions.

While Airbitz keeps things simple for the beginner it sacrifices nothing in terms of advanced bitcoin wallet features. Airbitz truely is easy enough that you can introduce bitcoin to friends and family in 15 minutes or less while maintaining the latest most empowering features for advanced users.

Safety First Architecture

Using bitcoin wallets can be a risky to users unless certain precautions are taken such as encrypting and backing up the private keys in their wallets. When using Airbitz your wallets are backed up and encrypted along with all transaction meta data instantly and automatically when you create your account.

Airbitz takes the risk (due to mobile device loss, theft, or damage) of losing your bitcoins out of the equation. If you lose or damage your device with bitcoins on it you can use any Android or iOS device with Airbitz installed to access your Airbitz wallet without having to restore any backups or deal with any encryption, keys, or files.

Safe, Simple, and secure

  • Wallet keys and data always encrypted client side and never leave your device unencrypted
  • Encrypted wallet data backed up to cloud automatically for you
  • Wallet restoration is simple just log into Airbitz on any compatible Android or iOS device and your data will be restored

Simple User Interface

Airbitz mobile app has been designed with the intentions that anyone who has used online banking should easily be able to use bitcoin in under 15 minutes.

  • Simple account creation requires just a login, password, and 4 digit PIN (How to create a bitcoin wallet video)
  • Automatic wallet encryption, backup, and multi-device synchronization with no configurations necessary
  • Receive or Send bitcoin using:
    • Email
    • SMS
    • QR Code
    • NFC (Near Field Communication)
    • BLE (Bluetooth Low Energy)

Secure Data Encryption

Airbitz takes security very seriously and it is built into the core or Airbitz. You can count on same level of security to be provided no matter what device or operating system Airbitz is running on.

  • Native C/C++ code for extremely strong encryption and password hashing (S-Crypt password hashing & AES256 encryption)
  • Local client-side encryption using user generated login & password of all data
  • No unencrypted wallet or transaction data stored on device

Learn more about bitcoin wallet security.

Private and Decentralized Architecture

The core principles behind Airbitz have led us to design the server software architecture to be one of the most private and decentralized bitcoin wallets available.

If Airbitz were to cease to exist there is no risk to your bitcoin, the same cannot be said for many other wallets. With Airbitz all wallet functionality to send and receive is built into the app and requires no server interaction. The only functionality that requires Airbitz servers to be online is related to the business directory features and your automatic wallet backups.

  • Zero personal user information required to use wallet (name, address, email, and phone number are not involved or used)
  • Local client-side encryption (S-Crypt password hashing & AES256 encryption) of all user keys and data
  • Hierarchical Deterministic wallets with changing addresses per transaction
  • No identifying or correlatable information stored on any server

Learn more about bitcoin wallet architecture.

Advanced Features

Aside from being simple to use for the beginner bitcoin user Airbitz wallet does not lack any enhanced features that any bitcoin veteran would expect from a world class hot wallet.

  • Supports Bluetooth Low Energy (BLE) to transfer funds instead of scanning QR codes
  • Multiple wallets per account with simple user defined wallet names (ie. “Vacation Fund”)
  • Integration with the Airbitz Business directory to auto-complete business meta data for transactions
  • User editable transaction data (payee, category, fiat value, & notes)
  • Built in transaction search by payee, category, or notes
  • Quick UI to transfer funds between wallets in an account
  • Multiple fiat currency support. Assign a fiat currency per wallet
  • Archive wallets not in use
  • Export transaction data or keys in multiple formats
