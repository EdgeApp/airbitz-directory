The Airbitz Mobile Bitcoin App is built on a platform that provides edge security for digital assets in a mobile-friendly, cross-platform SDK. Available as a powerful, open source software library, the Airbitz Core (ABC) provides all the security functionality and ease-of-use of the Airbitz mobile application including:

  • Automatic client-side data encryption using users' credentials
  • Automatic backup to P2P server nodes
  • Automatic synchronization between usersâ€™ devices
  • One-Touch 2 factor authentication
  • Password recovery
  • Data revision history and rollback
  • Blockchain operations such as private key generation, transaction signing, and blockchain queries

ABC gives developers the ability to build software such as ATMs, POS solutions, games, and payroll services that are fully integrated with accounts from the Airbitz mobile app. The Airbitz mobile app is the first product utilizing this platform, and the potential for future integrations with other applications is vast. Any developer that wishes to secure sensitive user data in a private, zero-knowledge, zero-trust infrastructure will find Airbitz a powerful solution to their data security.

Decentralized

With our deep roots in decentralized technologies such as the blockchain and digital currencies, we immediately see the benefits in decentralizing the majority of our security infrastructure. Our level of decentralization allows users to still access their data and even execute blockchain operations even if Airbitz servers are down. This focus provides the type of resilience and fault tolerance unmatched by other data security platforms.

Private by Design

Security and privacy go hand in hand, and ABC provides incredibly high privacy to promote a fundamental disincentive to attackers of the system. No part of ABC collects sensitive user data such as names, emails, addresses or phone numbers. This disassociates the users of our platform from the potentially highly valuable data they may be trying to protect.
